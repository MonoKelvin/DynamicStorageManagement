# 操作系统课程设计——动态分区分配管理算法的模拟（DynamicStorageManagement）

### 有可视化界面
编程语言是C++，使用QT框架

### 支持的分配算法
支持FF、NF、BF、WF算法<br>

### 支持紧凑算法

### 时间模拟
通过QTimer计时器的时间溢出方法(timeout())模拟时间流逝

### 支持文件读写
使用自定义文件后缀.dsm，保存主要信息，可通过打开文件重复观看模拟过程
